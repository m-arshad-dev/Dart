    üß† Variables and Data Types in Dart¶

    Understanding variables and data types is the foundation of any programming language ‚Äî and Dart is no exception.
    Let‚Äôs explore what they are, how to use them, and how Dart makes them both powerful and simple.

    ü™ú Step 1: What Is a Variable?¶

    A variable is like a container that stores a value ‚Äî it gives data a name so you can use it in your program.

    üß∫ Variable = Label + Value

    üí° Example¶

    void main() {
  var name = 'Arshad';
  print(name);
}

    Output:

    Arshad

    Explanation:

    Part Meaning
    name Variable name
    'Arshad' Value assigned
    var Dart automatically infers the type (String in this case)

    üß© Step 2: Declaring Variables¶

    There are multiple ways to declare variables in Dart, each with a specific purpose.

    Keyword Description Example
    var Type inferred automatically var age = 20;
    dynamic Type can change later dynamic data = 'Hello'; data = 123;
    final Immutable after assignment (runtime constant) final city = 'Lahore';
    const Immutable at compile time const PI = 3.14;
    Explicit Type Declare with a specific type int year = 2025;

    ‚ö° Step 3: Understanding Dart Data Types¶

    Dart is statically typed, meaning every variable has a specific type ‚Äî even when Dart infers it for you.

    üî¢ Numbers¶

    Dart supports two main number types:

    • int ‚Üí for integers (whole numbers)
    • double ‚Üí for decimal numbers

    Example¶

    void main() {
  int age = 25;
  double price = 9.99;

  print('Age: $age');
  print('Price: $price');
}

    Output:

    Age: 25
Price: 9.99

    ‚úÖ You can also use num (can store both int and double):

    num total = 100.5;

    üßµ Strings¶

    Strings represent text.

    void main() {
  String name = 'Arshad';
  String greeting = 'Hello, $name!'; // String interpolation
  print(greeting);
}

    Output:

    Hello, Arshad!

    String Tips

    • Use ' ' or " " ‚Äî both work.
    • Use triple quotes (''' or """) for multi-line strings.
    • Use interpolation ($variable or ${expression}) for string combination.

    ‚öôÔ∏è Booleans (bool)¶

    Represents true or false values.

    void main() {
  bool isActive = true;
  bool isLoggedIn = false;

  print(isActive);
  print(isLoggedIn);
}

    Output:

    true
false

    ‚úÖ Used often in conditions:

    if (isActive) {
  print('User is active!');
}

    üß∫ Lists (Arrays)¶

    A List stores multiple values in an ordered sequence.

    void main() {
  List<String> fruits = ['Apple', 'Banana', 'Mango'];
  print(fruits);
  print(fruits[1]); // Access by index
}

    Output:

    [Apple, Banana, Mango]
Banana

    ‚úÖ Modify lists dynamically:

    fruits.add('Orange');
fruits.remove('Banana');

    Note

    Lists in Dart are zero-indexed ‚Äî meaning the first element is at index 0.

    üó∫Ô∏è Maps (Key‚ÄìValue Pairs)¶

    Maps store data as key ‚Üí value pairs, similar to dictionaries in other languages.

    void main() {
  Map<String, String> user = {
    'name': 'Arshad',
    'city': 'Lahore',
    'country': 'Pakistan',
  };

  print(user['name']); // Access by key
}

    Output:

    Arshad

    Tip

    Maps are excellent for representing structured data ‚Äî such as user profiles or configuration settings.

    üî¢ Sets¶

    A Set is an unordered collection of unique values (no duplicates).

    void main() {
  Set<int> numbers = {1, 2, 3, 3, 2};
  print(numbers);
}

    Output:

    {1, 2, 3}

    üß± Step 4: Type Inference ‚Äî var vs dynamic¶

    var¶

    var name = 'Arshad';  // inferred as String
name = 'Ali';         // ‚úÖ allowed
// name = 10;         // ‚ùå Error: cannot assign int to String

    dynamic¶

    dynamic variable = 'Hello';
variable = 123;       // ‚úÖ Works
variable = true;      // ‚úÖ Works

    Tip

    Use: - var ‚Üí when type does not change.
    - dynamic ‚Üí when type may change during runtime.

    üîí Step 5: Constants ‚Äî final vs const¶

    Both create immutable variables.

    Keyword When to Use Example
    final Value known at runtime final now = DateTime.now();
    const Value known at compile-time const pi = 3.1416;

    Example¶

    void main() {
  final today = DateTime.now(); // runtime constant
  const version = 1.0;          // compile-time constant
}

    üîÑ Step 6: Type Conversion¶

    Convert between types easily:

    void main() {
  // String ‚Üí int
  int a = int.parse('10');

  // String ‚Üí double
  double b = double.parse('12.5');

  // int ‚Üí String
  String c = a.toString();

  // double ‚Üí int
  int d = b.toInt();

  print('$a, $b, $c, $d');
}

    Output:

    10, 12.5, 10, 12

    üìö Step 7: Summary Table¶

    Type Description Example
    int Whole numbers int age = 20;
    double Decimal numbers double price = 10.5;
    String Text String name = 'Dart';
    bool True/False bool isOk = true;
    List Ordered collection List<int> nums = [1, 2, 3];
    Map Key-value pairs Map user = {'id': 1};
    Set Unique values Set<int> s = {1, 2, 3};
    var Type inferred automatically var name = 'Ali';
    dynamic Type flexible dynamic x = 10; x = 'Hi';
    final Immutable (runtime) final city = 'Lahore';
    const Immutable (compile time) const PI = 3.14;

    üß† Quick Recap¶

    • Dart automatically infers variable types with var.
    • Use dynamic for variables that can hold different types.
    • Use final or const for immutable values.
    • Core built-in types: int, double, String, bool, List, Map, and Set.
    • Everything in Dart is an object, even numbers and booleans.
