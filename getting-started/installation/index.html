    üêß Installing Dart on Debian 13 (Trixie)¶

    Note

    üß† Debian 13 uses updated package security ‚Äî the old apt-key method is deprecated.
    Follow these new, secure steps to install the Dart SDK.

    Step 1 ‚Äî Update Your System¶

    sudo apt update && sudo apt upgrade -y
sudo apt install apt-transport-https curl gnupg -y

    Step 2 ‚Äî Add the Google Linux Repository Key¶

    sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://dl-ssl.google.com/linux/linux_signing_key.pub \
  | sudo gpg --dearmor -o /etc/apt/keyrings/dart.gpg

    Step 3 ‚Äî Add the Dart Repository¶

    echo "deb [signed-by=/etc/apt/keyrings/dart.gpg] \
https://storage.googleapis.com/download.dartlang.org/linux/debian stable main" \
| sudo tee /etc/apt/sources.list.d/dart_stable.list

    Step 4 ‚Äî Install Dart SDK¶

    sudo apt update
sudo apt install dart -y

    Step 5 ‚Äî Verify Installation¶

    dart --version

    Expected output:

    Dart SDK version: 3.x.x (stable)

    (Optional) Step 6 ‚Äî Add Dart to PATH¶

    If Dart isn‚Äôt recognized globally:

    echo 'export PATH="$PATH:/usr/lib/dart/bin"' >> ~/.bashrc

    Step 7 ‚Äî Test Your Setup¶

    Create and run a test file:

    mkdir ~/dart_projects

    Add the following code:

  print('Hello, Debian 13! Dart is working perfectly.');

    Run it:

    You should see:

    Success

    ‚úÖ Dart SDK is now fully installed and configured on Debian 13 (Trixie).
